YouTube Glass Feed

YouTube through glass!

Download this project as a .zip file Download this project as a tar.gz file

This small service-based app forwards YouTube video feeds to your Google Glass' timeline. From there, the videos will actually be viewable on Glass!

Following are some instructions for simple installation without having to build anything.

What you will need:

  • Google Glass
  • Android Development Kit
  • Android ADB USB Drivers (or equivalent for your OS of choice)

    Installation Instructions:
  • Download the two APK files you will need:
    http://www.androidfilehost.com/?fid=22962010882834654
    http://www.androidfilehost.com/?fid=22962010882834650
  • Open .youtubefeedconfig from the first download in your favorite text editor. Follow the instructions to configure what YouTube feed the app will fetch.
  • Plug in your device and open up an terminal window with access to adb. Type 'adb devices' to verify your Glass is found and ready.
  • Install the configuration file in the sdcard/ directory by executing this command: adb push .youtubefeedconfig sdcard/
  • Install the YouTube app from the second download:
    adb install com.google.android.youtube-2.apk
  • Install the Glass Feed app from the first download:
    adb install youtubeglassfeed.apk
  • Launch the app using Launchy OR Reboot your Glass and the app will automatically launch.
  • Watch your timeline and enjoy!

    How to uninstall:
  • Connect your glass via USB and run the following commands:
    adb uninstall com.appliedanalog.glass.youtube
    adb uninstall com.google.android.youtube
